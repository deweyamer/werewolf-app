# 快速开始指南 ⚡

## 🎯 5分钟开始你的第一局游戏

### 步骤1：启动服务（1分钟）

打开两个终端窗口：

**终端1 - 启动后端**
```bash
cd werewolf-app/server
npm run dev
```

**终端2 - 启动前端**
```bash
cd werewolf-app/client
npm run dev
```

看到以下信息表示启动成功：
```
🎮 Werewolf Server is running on port 3001
```

### 步骤2：上帝创建房间（1分钟）

1. 打开浏览器，访问：http://localhost:3000
2. 登录上帝账号：
   - 用户名：`god`
   - 密码：`god`
3. 点击"创建房间"
4. **记住房间码**（例如：ABC123）

### 步骤3：玩家加入房间（2分钟）

打开12个浏览器标签页，依次登录以下账号：

| 标签 | 用户名 | 密码 | 操作 |
|-----|--------|------|------|
| 1 | `test1` | `test` | 输入房间码 → 加入 |
| 2 | `test2` | `test` | 输入房间码 → 加入 |
| 3 | `test3` | `test` | 输入房间码 → 加入 |
| ... | ... | ... | ... |
| 12 | `test12` | `test` | 输入房间码 → 加入 |

**提示**：可以用 Ctrl+T（新标签页）和 Ctrl+W（关闭标签页）快速操作

### 步骤4：开始游戏（1分钟）

回到上帝的标签页：

1. ✅ 确认12名玩家全部加入
2. 点击"分配角色"
3. 点击"🎲 随机分配"
4. 点击"确认分配"
5. 点击"开始游戏"

**恭喜！游戏开始了！** 🎉

## 🎮 游戏流程速查

### 夜间阶段（上帝依次点击"进入下一阶段"）

1. **恐惧** → 噩梦之影选目标
2. **守护** → 摄梦人选目标
3. **狼刀** → 狼人选目标
4. **女巫** → 女巫选择救/毒/不使用
5. **预言** → 预言家选目标
6. **结算** → 公布死亡

### 白天阶段

7. **警长竞选**（仅第一天）
8. **投票放逐**
9. **猎人开枪**（如果猎人死亡）
10. **白天结算** → 进入下一回合

## ⚠️ 常见问题

### Q: 如何在一台电脑上测试？
A: 打开12个浏览器标签页，每个标签登录不同的test账号。

### Q: 玩家看得到其他人的角色吗？
A: 看不到。每个玩家只知道自己的角色。只有上帝能看到所有人的角色。

### Q: 游戏会自动进入下一阶段吗？
A: 不会。上帝需要手动点击"进入下一阶段"按钮。

### Q: 角色分配后可以修改吗？
A: 不可以。如需重新分配，请创建新房间。

### Q: 如何查看游戏记录？
A: 上帝控制台会显示最近的操作日志。
